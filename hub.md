## Description

Overrides core Obsidian color variables with a cozy pink palette (primary pink, soft pink, peach, lavender, mint). Rounds corners to 16px, softens borders, and reworks hover states so the whole app feels like a pastel notebook. Affects the sidebar, tabs, callouts, tags, and headings.

## Installation

1. Download `.obsidian/snippets/kawaii-pink.css` from this repo
2. Place it in your vault's `.obsidian/snippets/` folder
3. Enable it in Settings > Appearance > CSS Snippets
